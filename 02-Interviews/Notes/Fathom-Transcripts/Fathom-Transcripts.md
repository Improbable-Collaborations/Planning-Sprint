# Fathom Transcripts

Interview and meeting transcripts captured via Fathom. Each file contains the full transcript with a summary and action points at the top.

## File Naming

```
YYYY-MM-DD_Meeting-Title.md
```

For example: `2026-03-15_Kickoff-Call-With-OASIS.md`

## Template

```markdown
# Fathom Transcript — YYYY-MM-DD

- **Meeting:** Title
- **Date:** YYYY-MM-DD
- **Participants:** Names
- **Duration:** X min
- **Fathom link:** URL (if available)

## Summary

Brief summary of the meeting.

## Action Points

### Person Name
- [ ] Action item

### Unassigned
- [ ] Action item

## Full Transcript

Transcript text here.
```

Action points from these transcripts are consolidated into the daily action points file at `__STARTHERE/Action-Points/`.
