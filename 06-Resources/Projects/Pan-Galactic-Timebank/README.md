# Pan Galactic Timebank (PGT)

**Hitchhikers' Pan-galactic timebank built on OASIS.** Time exchanges, balance, ledger, onboarding, find-in-network; later clearing, dynamic equity (Slicing the Pie), OASIS attestation. Standalone Next.js app; reuses patterns from pangea-repo (Launchboard).
