# Voice Transcripts

Transcriptions of voice messages from team members across Telegram, Signal, WhatsApp and other channels. Each file contains the raw transcript with a summary and action points at the top.

## File Naming

```
YYYY-MM-DD_Speaker-Name_Topic.md
```

For example: `2026-03-15_David-Bovill_Auth-Middleware.md`

## Template

When adding a transcript, use this structure:

```markdown
# Voice Transcript — YYYY-MM-DD

- **From:** Name
- **Channel:** Telegram / Signal / WhatsApp
- **Date:** YYYY-MM-DD
- **Duration:** approx. X min

## Summary

Brief summary of the message.

## Action Points

- [ ] Action item one
- [ ] Action item two

## Full Transcript

Transcript text here.
```
