# Action Points

## Weekly Files (start here)

Each person has one file per week with all their action items. Open yours:

```
__STARTHERE/Action-Points/Weekly/2026-W12_Graham.md
__STARTHERE/Action-Points/Weekly/2026-W12_David.md
__STARTHERE/Action-Points/Weekly/2026-W12_Max.md
```

Unassigned items need an owner: `Weekly/2026-W12_Unassigned.md`

Tick items in Obsidian by clicking the checkbox. Add notes by indenting below the item.

## Daily Files (raw record)

Daily files in this folder are the source record, organised by date. The weekly files pull from these. No need to check daily files unless you want the full context for a specific day.

## How It Works

1. Voice transcripts and Fathom calls generate daily action point files
2. Weekly rollups are created per person from those daily files
3. Unchecked items carry forward to the next week
