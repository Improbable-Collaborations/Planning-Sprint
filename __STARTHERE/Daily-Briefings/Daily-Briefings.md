# Daily Briefings

Human-readable summaries of what happened each day — what changed, why, and what needs attention. Designed to be the first thing you read when you come online.

Each file is named `Briefing-YYYY-MM-DD.md`.

For the raw commit log, see [Change-Log](../Change-Log/Change-Log.md).
For consolidated action items, see [Action-Points](../Action-Points/Action-Points.md).
