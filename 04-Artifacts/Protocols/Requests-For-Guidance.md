## Purpose

Define a lightweight way to request guidance from the team — whether that's a decision, feedback, expertise, or a second pair of eyes — without creating dependency on any single person.

## Key Questions to Resolve

- Where do requests for guidance get posted?
- What information should a request include?
- What is the expected response time?
- How do we escalate if guidance isn't forthcoming?
- How do we distinguish between "I need a decision" and "I need advice"?
