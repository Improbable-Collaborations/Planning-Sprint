# Planning Sprint

A structured planning sprint repository for discovery, stakeholder alignment, and strategic planning.

## Structure

```
01-Discovery/           Stakeholder mapping, current-state assessment, constraints
02-Interviews/          Templates, notes, and synthesis from key player interviews
03-Meetings/            Kickoff, working sessions, standups, retrospectives
04-Artifacts/           Goals, roadmap, risk register, decision log, RACI matrix
05-Communications/      Status updates and presentations
06-Resources/           Reference materials and supporting documents
```

## Sprint Workflow

1. **Discovery** — Map stakeholders, assess current state, identify constraints
2. **Interviews** — Conduct structured interviews with key players
3. **Synthesis** — Consolidate findings into actionable insights
4. **Planning** — Define goals, build roadmap, assign ownership (RACI)
5. **Alignment** — Review with stakeholders, iterate, finalize

## Getting Started

Begin with `01-Discovery/Stakeholder-Mapping/` to identify key players, then use the interview templates in `02-Interviews/Templates/` to structure conversations.
